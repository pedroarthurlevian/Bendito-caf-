# Bendito Café v2

Versão aprimorada do site, com cardápio ampliado usando os nomes do PDF enviado.

## Recursos
- visual rústico premium em verde
- cardápio completo com todos os produtos do PDF organizados por categoria
- filtros por categoria e subcategoria
- busca e ordenação
- carrinho funcional com cupom `BENDITO10`
- favoritos
- reservas com persistência em localStorage
- newsletter
- FAQ interativo
- galeria com lightbox
- depoimentos em slider
- localização com mapa

## Como usar
1. Extraia o `.zip`
2. Abra a pasta no VS Code
3. Abra `index.html` no navegador ou use Live Server

## Observação
Tudo funciona no front-end. Pedidos, favoritos, newsletter e reservas ficam salvos no navegador para simulação real.
